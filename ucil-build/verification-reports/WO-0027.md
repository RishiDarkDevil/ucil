## Root cause

Cross-module `PATH`-mutation race — detailed in
`ucil-build/rejections/WO-0027.md` ➝ "Root cause". The four
`watcher::test_*` tests that exercise `detect_watchman` /
`auto_select_backend` set a process-global `PATH` and serialize only
against each other via a module-local `ENV_GUARD` Mutex, missing the
cross-module leak to `session_manager::tests::*` and
`plugin_manager::test_hot_cold_lifecycle`.

`cargo nextest` (per-test process isolation) masks this from the
executor's local check; `cargo llvm-cov` drives `cargo test` directly
(single libtest process with thread-per-test) and exposes it. The
coverage gate has been the canonical catch for this failure mode
since Phase 0.
