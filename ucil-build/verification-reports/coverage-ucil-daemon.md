## Why the auto-script fails

The auto-runner `scripts/verify/coverage-gate.sh` exits 1 because the
`cargo llvm-cov` wrapper does not generate `.profraw` files when
`RUSTC_WRAPPER=sccache` is in the inherited environment (the wrapper
delegation chain breaks). The fix is to invoke `cargo llvm-cov`
directly with `RUSTC_WRAPPER` unset:

```
cd /home/rishidarkdevil/Desktop/ucil-wt/WO-0048
env -u RUSTC_WRAPPER cargo llvm-cov clean --workspace
env -u RUSTC_WRAPPER cargo llvm-cov --package ucil-daemon --summary-only
```

Carry-forward from WO-0042 / 0043 / 0044 / 0045 / 0046 / 0047 and now
WO-0048 (11th consecutive). The standing escalation
`20260419-0152-monitor-phase1-gate-red-integration-gaps.md` (now
archived to `escalations/archive/`) tracks the long-term fix; the
manual workaround is the contractual path until that lands.
