### Warnings (should fix; non-blocking if justification exists)

#### W1. `pub const VERSION` in `lib.rs` is outside WO-0002 scope

`crates/ucil-core/src/lib.rs:15`

```rust
/// Crate version, identical to the `Cargo.toml` package version.
pub const VERSION: &str = env!("CARGO_PKG_VERSION");
```

WO-0002's `scope_in` for `lib.rs` specifies only `pub mod` declarations and
`pub use` re-exports.  The Rust style rules state *"lib.rs is only re-exports
and module declarations — no logic."*  A `pub const` is neither a module
declaration nor a re-export.

This is a minor, benign addition (it is documented, correct, and harmless), but
it technically violates the scope boundary.  Either remove it, move it to a
separate `version.rs` submodule in a later WO, or create an ADR accepting it.
