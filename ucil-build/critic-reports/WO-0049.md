4. **`SerenaG1Source` production mapping diverges from the test-factory's Serena payload shape.**
   The production `SerenaG1Source::execute` at `g1_sources.rs:379-447` maps a `Ok(Some(HoverDoc { markdown, source }))` response to a single `G1FusionEntry` with fields `{ "usage_type": "definition", "hover_doc": <markdown>, "source": <"serena"|"lsp"|"none"> }` — no `signature` field, `usage_type` always `"definition"`. This is acceptable for a unit test that proves the orchestration shape; the verifier should note that a future integration test driving `SerenaG1Source` against a real `SerenaHoverClient` would close the gap.

6. **`tokio::task::block_in_place` requires a multi-thread runtime at handler call sites.**
