**File**: `crates/ucil-cli/src/commands/init.rs:178–182`

**Expected**: Every `.await` that touches IO wrapped in `tokio::time::timeout` per the
project-wide invariant in `CLAUDE.md`:
> All async code uses `tokio::time::timeout` on any await that touches IO.

**Actual**: The `.output().await` for each P0 binary probe has no timeout wrapper:

```rust
// lines 178-182

    .arg("--version")
    .output()   // ← no tokio::time::timeout wrapping this await
    .await
{
    Ok(out) if out.status.success() => PluginStatusKind::Ok,
    _ => PluginStatusKind::Degraded,
};
```

`shellcheck`) hangs on `--version`, `ucil init` will block indefinitely.
